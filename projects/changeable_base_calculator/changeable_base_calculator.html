Changeable Base Calculator

Overview

Most programmers are familiar with a few base systems outside of our standard base 10 system, such as base 2 (binary), base 16 (hex), and even base 8 (octal). For an honors project in 9th grade, I wrote a simple calculator which could function in any base system. Ever wanted to know what PI looks like in base 13? Well now you can.

This was an early Python project for me, and I learned a lot about the language with it. I also learned a lot about fundamental arithmetic, bridging two of my passions, programming and mathematics. I would like to work on more projects like this in the future.
