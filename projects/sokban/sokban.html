soKBan

Overview

Much like my other project Bewof, soKBan is an attempt to compact an entire game into only 1024 bytes of C source code. Having learned from Bewof, I was able to implement a fully functioning Sokoban game, but an entire level generation system as well (which generates levels that are guaranteed to be solvable). I was quite proud of the way I compacted the level generation code into the game code, carefully weaving the two together to save on space.

Like Bewof, I have also included the compleat source code right below:
