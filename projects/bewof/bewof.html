Bewof

Overview

During my early years as a programer, I become very interested in classic, ascii based roguelikes. My search for cool ones to play eventually led me to a game called Zigclimb, a simple, slick looking roguelike. The wiki mentioned the source code for the game was available, but all I could see was this random jumble of text. It took me a few moments to realize that was the Zigclimb source code. 1024 bytes of C source code, representing the entire game. I was compleat blown away. How could so much complexity be squished down to such a small footprint. I needed to find out.

Several weeks later, after thoroughly dissecting the Zigclimb source code and fighting the gcc compiler, I had achieved my own feat. A 1024 byte roguelike, compleat with fancy new mechanics, and even a whole optional class system. This project is one of my pride and joys, and I am happy to present the entire source code in all its glory, right here:
